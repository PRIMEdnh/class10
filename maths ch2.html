Long Answer Questions (NCERT)

  1. Factorise x³ + 3x² - x - 3.
    Answer(x³ + 3x²) - (x + 3) = x²(x + 3) - 1(x + 3) = (x² - 1)(x + 3) = (x - 1)(x + 1)(x + 3)
  2. Divide x³ + 2x² - x - 2 by x + 2.
    AnswerQuotient = x² - 1, Remainder = 0
  3. Factorise x³ - 6x² + 11x - 6.
    Answer(x - 1)(x - 2)(x - 3)
  4. Find zeroes of x³ - 3x² - 4x + 12.
    Answer2, -1, 3
  5. Divide 2x³ + 3x² - 11x - 6 by x + 2.
    AnswerQuotient = 2x² - x - 3, Remainder = 0
  6. Factorise x³ + x² - 4x - 4.
    Answer(x² - 4)(x + 1) = (x - 2)(x + 2)(x + 1)
  7. Check whether x³ + 8 is factorable using sum of cubes formula.
    Answer(x + 2)(x² - 2x + 4)
  8. Factorise x³ - 27 using difference of cubes.
    Answer(x - 3)(x² + 3x + 9)
  9. Find quotient and remainder when x³ + 4x² + x - 6 is divided by x + 2.
    AnswerQuotient = x² + 2x - 3, Remainder = 0
  10. Divide 3x³ - 5x² + 2x - 4 by x - 1.
    AnswerQuotient = 3x² - 2x + 0, Remainder = -4
