Very Short Answer Questions

  1. Write the general form of a quadratic equation.
    Answerax² + bx + c = 0, where a ≠ 0
  2. Give an example of a quadratic equation.
    Answerx² - 5x + 6 = 0
  3. Define discriminant.
    AnswerThe value b² - 4ac of a quadratic equation ax² + bx + c = 0
  4. When does a quadratic equation have real and distinct roots?
    AnswerWhen discriminant > 0
  5. When does a quadratic equation have no real roots?
    AnswerWhen discriminant < 0
  6. Write the roots of x² - x - 6 = 0.
    Answer3, -2
  7. Write the sum of roots of x² + 7x + 10 = 0.
    Answer-7
  8. Write the product of roots of x² - 8x + 15 = 0.
    Answer15
  9. Write roots of x² + 4x + 4 = 0.
    Answer-2, -2
  10. Factorise x² - 5x + 6.
    Answer(x - 2)(x - 3)
  11. Solve x² - 6x + 8 = 0.
    Answer(x - 2)(x - 4) = 0 → x = 2, 4
  12. Solve 2x² - x - 6 = 0.
    Answer(2x + 3)(x - 2) = 0 → x = -3/2, 2
  13. Solve x² + 5x + 6 = 0.
    Answer(x + 2)(x + 3) = 0 → x = -2, -3
  14. Solve x² - 2x - 15 = 0.
    Answer(x - 5)(x + 3) = 0 → x = 5, -3
  15. Write discriminant of 3x² - 2x - 1 = 0.
    AnswerDiscriminant = (-2)² - 4*3*(-1) = 4 + 12 = 16
